Scrapper Project - Scraping CNBC News Headlines

This Python script scrapes the latest headlines from CNBC.

How to use it locally:

  1. Clone the repository: 
    git clone https://github.com/amireshhhh/Scrapper.git
  2. Navigate to the repository directory: 
    cd Scrapper
  3. Install dependencies: 
    pip install -r requirements.txt
  4. Run the scraper: 
    python scrapper.py

For more information, visit the GitHub repository.
