Docker News Scraper Project

This project scrapes news headlines from Bloomberg, CNBC, WSJ, Benzinga, and Yahoo Finance using Python, BeautifulSoup, and Docker. You can find the full code and instructions on how to run it in the GitHub repository.

How It Works

The project scrapes news headlines from various websites, stores them in a CSV file, and serves them via a Flask web application. The code is containerized using Docker, ensuring that it can run consistently across different environments.

Technologies Used

  • Python (Requests, BeautifulSoup)
  • Pandas for data storage
  • Flask for web application
  • Docker for containerization

How to Run the Project Locally

  1. Clone the project from GitHub
  2. Ensure you have Docker installed on your system.
  3. Navigate to the project directory and build the Docker container using the following command:
    4. docker build -t my-scrapper .
  4. Run the container:
    5. docker run -p 5000:5000 my-scrapper
  5. Open your browser and go to http://localhost:5000 to view the latest headlines!
